# 精简 Prompt 模板

按需复制并替换方括号。模板是参考，不应逐项拼接；用户已给出具体描述时保留其原意。

## 通用结构

```text
用途：[资产用途]
任务类型：[分类 slug]
核心请求：[用户主要目标]
输入图片：[图一：角色；图二：角色]（可选）
场景/背景：[环境]
主体：[主体和动作]
风格/媒介：[摄影 / 插画 / 3D]
构图/景别：[画幅、视角、焦点、留白]
光线/氛围：[光源、时间、情绪]
文字（逐字）："[精确文字]"
必须保持：[不变项]
禁止：[额外主体、文字、Logo、水印、畸形、重复元素]
```

## 生成

### 写实场景

```text
用途：编辑配图
任务类型：photorealistic-natural
核心请求：生成一张写实照片：[主体]正在[行为]，位于[环境]。
构图/景别：[景别和视角]；[需要时保留留白]
光线/氛围：[自然光线]，自然色彩和景深。
色彩/材质：真实的[关键材质和纹理]。
禁止：过度磨皮、棚拍质感、额外人物、文字、Logo、水印。
```

### 商品摄影

```text
用途：商品详情页主图
任务类型：product-mockup
核心请求：生成[产品]的高端商品照片。
场景/背景：[背景]。
主体：[材质和颜色]的产品，轮廓完整，标签和结构清晰。
构图/景别：[视角]，保留充足边距。
光线/氛围：使用[灯光]表现材质。
禁止：未提供的品牌、装饰物、文字、水印。
```

### 信息图 / 教学图

```text
用途：[演示文稿 / 教学海报]
任务类型：infographic-diagram
核心请求：为[受众]制作一张[主题]信息图，按[方向]展示：[步骤或模块]。
构图/景别：使用清晰箭头、统一图标和强层级，背景简洁。
文字（逐字）："[所有需要出现的文字]"
禁止：其他标签、Logo、水印、虚构数据。
```

### UI 线框图

```text
用途：网站线框图
任务类型：ui-mockup
核心请求：[页面或流程]布局。
风格/媒介：低保真灰度 wireframe。
主体：[按顺序列出导航、主视觉、功能卡、表格、页脚等模块]。
构图/景别：[桌面横向 / 移动端纵向]。
禁止：颜色、真实照片、品牌 Logo、水印。
```

### 风格化插画

```text
用途：[封面 / 叙事插图]
任务类型：illustration-story
核心请求：绘制[主体和行为]，场景为[环境]。
风格/媒介：[风格、媒介和笔触]。
色彩/材质：以[主色]为主。
光线/氛围：[光线与氛围]。
构图/景别：[构图]。
禁止：新故事元素、文字、Logo、水印。
```

### 历史场景

```text
用途：历史文章配图
任务类型：historical-scene
核心请求：呈现[地点]在[年代/日期]的[事件或日常场景]。
主体：[人物、服装、道具]。
场景/背景：[符合时代的建筑和环境]。
必须保持：服装、道具、建筑与时代准确一致。
禁止：现代物品、文字、Logo、水印。
```

## 编辑与融合

### 人物一致性

```text
用途：[试穿图 / 人物修图]
任务类型：identity-preserve
输入图片：图一：编辑目标；图二：服装参考。
核心请求：只将图一人物的[指定元素]改为图二中的[目标效果]。
必须保持：面部身份、体型、姿势、发型、表情、背景、镜头和构图不变。
禁止：新增配饰、文字、水印。
```

### 精确移除或替换

```text
任务类型：precise-object-edit
输入图片：图一：编辑目标。
核心请求：只移除/替换图一中的[对象和位置]，目标为[新内容或自然补全]。
必须保持：周围纹理、结构、光线、阴影和所有其他物体不变。
```

### 文字本地化

```text
任务类型：text-localization
输入图片：图一：原始图片。
核心请求：只把图一中的“[原文]”替换为“[目标文字]”。
必须保持：布局、字体风格、字号、颜色、位置、对齐、间距和其他图像内容不变。
禁止：额外文字、改动 Logo、重排版。
```

### 光照或天气

```text
任务类型：lighting-weather
输入图片：图一：原始照片。
核心请求：只将画面改为[冬日晚间 / 雨后清晨等]。
必须保持：主体身份、几何、镜头、构图和物体位置不变；只改变光照、氛围和天气。
```

### 风格迁移

```text
任务类型：style-transfer
输入图片：图一：风格参考。
核心请求：以图一的视觉风格生成/修改[目标内容]。
必须保持：图一的配色、纹理、笔触和光影语言；不要复制图一的具体主体或文字。
禁止：额外元素。
```

### 多图融合

```text
任务类型：compositing
输入图片：图一：基础场景；图二：[人物/服装/物体]来源。
核心请求：将图二的[内容]自然放入图一的[位置]。
必须保持：图一构图和未编辑区域不变；匹配光线、透视、比例、清晰度、遮挡和接触阴影。
禁止：额外元素。
```

### 草图转写实

```text
任务类型：sketch-to-render
输入图片：图一：草图。
核心请求：将图一草图转为[写实照片 / 指定渲染风格]。
必须保持：布局、比例、轮廓和透视；使用合理材质和光线补全细节。
禁止：草图中不存在的物体或文字。
```

## Pro 交互标记

### 圈选区域编辑

```text
任务类型：precise-object-edit
输入图片：图一：带圈选标记的编辑目标。
核心请求：在左下角圈选区添加[对象一]，在右侧标记区添加[对象二]。
必须保持：未标记区域、原构图、透视和光线不变；新增对象有自然接触阴影。
禁止：最终图中的圆圈、箭头、草图线、辅助点和水印。
```

### 坐标对应替换

```text
任务类型：precise-object-edit
输入图片：图一：带编号标记的编辑目标。
核心请求：将每个来源物体放到人物手中相同编号的位置，严格遵循编号对应关系。
必须保持：人物身份、姿势、手部、背景和未标记物体不变。
禁止：全部编号和辅助标记。
```

## Lite 组图与联网

### 连贯组图

```text
任务类型：illustration-story
核心请求：生成恰好[数量]张内容关联的图片：同一[主体]在同一[场景]中依次经历[变化列表]。
必须保持：主体身份、服装、关键道具、镜头方向和视觉风格一致；每张都是独立完整画面。
禁止：拼贴、联系表、标题、Logo、水印。
```

### 联网事实信息图

```text
任务类型：infographic-diagram
核心请求：使用联网搜索核实[地点/主题/日期范围]的最新信息，制作[横向/纵向]信息图。
构图/景别：按[排序规则]展示[字段列表]，缺失信息不要编造。
文字（逐字）："[逐字标题]"
禁止：未经核实的数据、额外标签、Logo、水印。
```

## 透明背景

```text
任务类型：background-extraction
核心请求：将[主体]完整、独立地置于纯色 [KEY_COLOR]（[HEX]）背景中央。
主体：轮廓完整、边缘清晰；与画布四边完全分离，保留充足 padding；不得裁切或触边。
场景/背景：整张画布为单一、完全均匀、无压缩噪点的 [KEY_COLOR]；无渐变、纹理、地面、背景物体或光照变化。
必须保持：[用户明确要求保留的材质、细节、标签文字或 Logo]；主体不得出现 [KEY_COLOR] 及其相邻色相族。
禁止：cast/contact shadow、地面反射、背景反射、key-colored rim light、glow、ambient bounce、motion blur、halo、fringing、水印和未要求的文字/Logo。
```

用户未要求保留标签或 Logo 时才禁止额外文字/Logo。玻璃、金属可保留固有材质高光，但通常不适合简单色键；毛发、烟雾、液体、薄纱、运动模糊、软阴影和半透明对象应停止此方案。
