# 视觉参考示例

仅在用户明确要求相应视觉方向、需要查看典型效果或希望将示例作为风格参考时读取本文。不要默认把这些图片加入请求，不要照搬示例中的主体、场景、构图或文字；用户没有授权参考图时，仅把它们用于理解分类。

需要实际作为模型输入时，按用户选择把对应文件传给 `--image`，并在 prompt 中将其标为「风格参考」。示例不代表 Seedream 的固定输出质量，也不替代生成后验收。

## 写实自然场景

- 分类：`photorealistic-natural`
- 文件：[photorealistic-natural.png](../assets/examples/photorealistic-natural.png)
- 适用：自然光、纪实人物、真实皮肤与环境材质。
- 限制：只借鉴写实程度、光线和材质；不要默认添加园艺师、屋顶花园或城市天际线。

```text
横向纪实摄影参考图：清晨自然光下，一位成年园艺师在城市屋顶花园照料香草。真实皮肤、布料、叶片、混凝土与木材质感，人物自然不摆拍，中远景，城市天际线轻微虚化，绿色与暖阳为主。无文字、无品牌、无 logo、无水印、无边框、非拼贴。
```

## 流程信息图

- 分类：`infographic-diagram`
- 文件：[infographic-diagram.png](../assets/examples/infographic-diagram.png)
- 适用：步骤流程、图标与短标签、清晰层级和横向信息组织。
- 限制：只借鉴信息层级与版式；文字必须按当前任务逐字提供，不得沿用本图文案。

```text
横向中文信息图参考图，标题不需要。画面中央从左到右只有四个清晰步骤卡片，用细箭头依次连接。四张卡片上的文字必须逐字、仅为：“灵感”“草图”“生成”“交付”，每张卡片一个词，顺序严格一致，不添加任何其他文字。对应图标依次为发光灯泡、铅笔线稿、生成中的彩色图像、打勾的成品文件。现代编辑设计，暖白背景，深灰文字，珊瑚橙、蓝绿色和柔黄作强调，清楚的层级和充足留白。无品牌、无 logo、无水印、无边框、非拼贴。
```
